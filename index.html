Introduction to jQuery

These examples should provide students a basic understanding of including and using jQuery in projects.
They should also gain insight into the power of jQuery.

Download and Use jQuery

Direct Download

Download from jquery.com/download, copy this file into your JS folder and add the script below to the bottom of your body tag.

<script src="js/jquery-1.11.2.min.js"></script>

Using jQuery CDN ( content delivery network )

The jQuery CDN is something that should be used when a site is going to be loaded onto the server i.e. going live, this will allow for a faster server call, rather than having to search through your files for jQuery you can connect directly by including the script below at the bottom of your code above the closing</body> tag

<script src="https://code.jquery.com/jquery-1.11.2.min.js"></script>

Using Google's Hosted jQuery CDN Library

This is the same as the CDN above, only using Google's Hosted Library - this is the third way you can include jQuery and should be used in the live version if not using the jQuery hosted CDN.

<script src="https://ajax.googleapis.com/ajax/libs/jquery/1.11.3/jquery.min.js"></script>

Using jQuery to select HTML elements

Select all p tags & hide them

$('p').hide();

Select the test id and hide the element with this ID

$('#test').hide();

Select the test class and hide any elements with this class

$('.test').hide();
